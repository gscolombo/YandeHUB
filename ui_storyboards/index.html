Projeto de interface de usuário

Abaixo são apresentados os storyboards para representação da interface de usuário. Todo storyboard é composto por wireframes das interfaces de usuário, criadas utilizando o aplicativo Figma. O projeto no aplicativo contendo os wireframes e os protótipos de tela podem ser acessados por aqui.
Os storyboards foram criados para cada seção dos requisitos funcionais, utilizando o detalhamento de cada história de usuário para orientar a criação dos wireframes e as transições associadas.
